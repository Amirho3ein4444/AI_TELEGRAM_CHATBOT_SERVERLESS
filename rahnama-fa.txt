راهنمای ساخت ربات چت رایگان به زبان فارسی
سلام! توی این راهنما می‌خوام بهت یاد بدم چطور یه ربات چت بسازی که مجانی باشه و بتونه با آدما توی تلگرام حرف بزنه. این ربات از Appwrite Cloud برای نگه‌داری اطلاعات و اجرای برنامه استفاده می‌کنه، از OpenRouter برای جواب‌های هوشمندش کمک می‌گیره و توی تلگرام کار می‌کنه. همه‌چیز رو قدم‌به‌قدم توضیح می‌دم، حتی اگه تا حالا چیزی از اینا نشنیده باشی!
چیزی که می‌سازیم
رباتت قراره:
با Appwrite Cloud اطلاعات رو نگه داره و برنامه‌ش اجرا بشه.

از OpenRouter جواب‌های باهوش و مجانی بگیره.

توی تلگرام با آدما حرف بزنه.

برنامه‌ی رباتت رو توی گیت‌هاب (یه سایت برای نگه‌داری کد) می‌ذاری و یه فایل به اسم main.js (که خودت داری) براش استفاده می‌کنی.
چیزایی که قبلش نیاز داری
فقط به چندتا چیز ساده نیاز داری:
یه کامپیوتر با اینترنت.

یه مرورگر وب (مثل کروم، فایرفاکس یا سافاری).

یه گوشی یا کامپیوتر که تلگرام روش نصب باشه (اگه نداری، از telegram.org دانلود کن).

یه ایمیل برای ثبت‌نام توی سایت‌ها.

همین کافیه! بریم شروع کنیم.
قدم ۱: ثبت‌نام توی Appwrite Cloud
Appwrite Cloud یه سرویس مجانیه که مثل مغز رباتت عمل می‌کنه. اطلاعات رو نگه می‌داره و برنامه‌ت رو اجرا می‌کنه.
مرورگر رو باز کن: مرورگرت رو باز کن (مثل گوگل کروم).

برو به Appwrite: توی نوار آدرس بنویس https://appwrite.io/ و دکمه Enter رو بزن.

ثبت‌نام کن: گوشه‌ی بالای سمت راست صفحه، یه دکمه به اسم Sign Up (ثبت‌نام) می‌بینی. روش کلیک کن.

یه راه انتخاب کن:
اگه حساب گوگل داری، روی Sign up with Google کلیک کن و با گوگل ثبت‌نام کن.

یا روی Sign up with Email کلیک کن، بعد ایمیلت و یه رمز بنویس.

ثبت‌نام رو تموم کن: مراحل رو دنبال کن (مثل نوشتن اسمت). اگه با ایمیل ثبت‌نام کردی، برو توی ایمیلت، یه لینک تأیید میاد، روش کلیک کن.

وارد شو: بعد از ثبت‌نام، برگرد به https://appwrite.io/، روی Log In (ورود) کلیک کن و ایمیل و رمزی که انتخاب کردی رو بنویس یا با گوگل وارد شو.

حالا توی صفحه‌ی اصلی Appwrite هستی! اینجا قراره رباتت رو بسازی.
قدم ۲: ساختن پروژه توی Appwrite Cloud
پروژه مثل یه جای کار برای رباتته توی Appwrite.
صفحه‌ی اصلی رو پیدا کن: وقتی وارد شدی، یه دکمه بزرگ وسط صفحه می‌بینی که نوشته Create Project (ساخت پروژه).

پروژه رو بساز: روی اون دکمه کلیک کن.

اسم بذار: یه پنجره باز می‌شه. توی کادر “Project Name” (اسم پروژه)، یه اسم ساده مثل ربات من بنویس. بعد دکمه‌ی آبی Create (ساختن) رو بزن.

صبر کن: Appwrite چند ثانیه طول می‌کشه تا پروژه‌ت رو آماده کنه. بعدش یه صفحه‌ی جدید می‌بینی که اسمش بالای صفحه نوشته شده.

پروژه‌ت آماده‌ست! اینجا همه‌چیز رباتت رو می‌ذاری.
قدم ۳: راه‌اندازی پایگاه داده توی Appwrite
رباتت نیاز داره اطلاعات رو یه جایی نگه داره، مثلاً کی داره باهاش حرف می‌زنه و چی گفته. توی Appwrite به این می‌گن پایگاه داده، و داخلش مجموعه‌ها (مثل پوشه‌ها) درست می‌کنیم. باید سه تا مجموعه بسازی: کاربرها، جلسه‌ها، و چت‌ها. هر کدوم رو دقیق برات توضیح می‌دم.
۳.۱ ساخت مجموعه‌ی "کاربرها"
اینجا اطلاعات کسایی که از رباتت استفاده می‌کنن رو نگه می‌داری.
برو به پایگاه داده: توی صفحه‌ی پروژه‌ت (ربات من)، سمت چپ یه آیکون به اسم Database (پایگاه داده) هست، مثل یه کمد کوچیک. روش کلیک کن.

یه مجموعه بساز: بالای صفحه، دکمه‌ی نارنجی Add Collection (اضافه کردن مجموعه) رو بزن.

اسمش رو بذار: توی کادر “Name” (اسم)، بنویس کاربرها (با “ک” بزرگ). بعد دکمه‌ی آبی Create (ساختن) رو بزن.

ویژگی‌ها رو اضافه کن: حالا توی مجموعه‌ی “کاربرها” هستی. باید بگی چه اطلاعاتی نگه داره. دکمه‌ی نارنجی Add Attribute (اضافه کردن ویژگی) رو بزن.
ویژگی ۱: telegramId
نوع: از منوی کشویی، String (رشته) رو انتخاب کن (برای متن).

کلید: بنویس telegramId (همه‌ش کوچیک).

اندازه: بنویس 255 (یعنی تا ۲۵۵ حرف جا داره).

لازمه؟: تیک “Required” (لازم) رو بزن (هر کاربر باید اینو داشته باشه).

دکمه‌ی آبی Create (ساختن) رو بزن.

ویژگی ۲: username
نوع: دوباره String رو انتخاب کن.

کلید: بنویس username (همه‌ش کوچیک).

اندازه: بنویس 255.

لازمه؟: تیک “Required” رو نزن (چون همه کاربرا اسم ندارن).

دکمه‌ی Create رو بزن.

ویژگی ۳: createdAt
نوع: DateTime (تاریخ و ساعت) رو انتخاب کن.

کلید: بنویس createdAt (با “c” کوچیک).

لازمه؟: تیک “Required” رو بزن.

دکمه‌ی Create رو بزن.

چک کن: باید سه تا ویژگی ببینی: telegramId, username, createdAt. اگه اشتباه کردی، کنارش یه سطل آشغال هست، روش کلیک کن و دوباره بساز.

۳.۲ ساخت مجموعه‌ی "جلسه‌ها"
اینجا می‌فهمی کی داره با رباتت چت می‌کنه.
برگرد عقب: بالای صفحه، روی Collections (مجموعه‌ها) کلیک کن.

یه مجموعه دیگه بساز: دکمه‌ی Add Collection رو بزن.

اسمش رو بذار: بنویس جلسه‌ها (با “ج” بزرگ) و Create رو بزن.

ویژگی‌ها رو اضافه کن: توی “جلسه‌ها”، دکمه‌ی Add Attribute رو بزن.
ویژگی ۱: userId
نوع: String رو انتخاب کن.

کلید: بنویس userId (با “u” کوچیک).

اندازه: بنویس 255.

لازمه؟: تیک “Required” رو بزن.

Create رو بزن.

ویژگی ۲: active
نوع: Boolean (درست/غلط) رو انتخاب کن.

کلید: بنویس active (همه‌ش کوچیک).

مقدار پیش‌فرض: بنویس true (کوچیک).

لازمه؟: تیک “Required” رو بزن.

Create رو بزن.

ویژگی ۳: createdAt
نوع: DateTime رو انتخاب کن.

کلید: بنویس createdAt.

لازمه؟: تیک “Required” رو بزن.

Create رو بزن.

چک کن: باید userId, active, createdAt رو ببینی.

۳.۳ ساخت مجموعه‌ی "چت‌ها"
اینجا پیام‌هایی که بین کاربر و ربات رد و بدل می‌شه رو نگه می‌داری.
برگرد عقب: روی Collections کلیک کن.

مجموعه بساز: دکمه‌ی Add Collection رو بزن.

اسمش رو بذار: بنویس چت‌ها (با “چ” بزرگ) و Create رو بزن.

ویژگی‌ها رو اضافه کن: دکمه‌ی Add Attribute رو بزن.
ویژگی ۱: sessionId
نوع: String رو انتخاب کن.

کلید: بنویس sessionId (با “s” کوچیک).

اندازه: بنویس 255.

لازمه؟: تیک “Required” رو بزن.

Create رو بزن.

ویژگی ۲: userId
نوع: String رو انتخاب کن.

کلید: بنویس userId.

اندازه: بنویس 255.

لازمه؟: تیک “Required” رو بزن.

Create رو بزن.

ویژگی ۳: role
نوع: String رو انتخاب کن.

کلید: بنویس role (همه‌ش کوچیک).

اندازه: بنویس 255.

لازمه؟: تیک “Required” رو بزن.

Create رو بزن.

ویژگی ۴: content
نوع: String رو انتخاب کن.

کلید: بنویس content (همه‌ش کوچیک).

اندازه: بنویس 10000 (برای پیام‌های بلند).

لازمه؟: تیک “Required” رو بزن.

Create رو بزن.

ویژگی ۵: createdAt
نوع: DateTime رو انتخاب کن.

کلید: بنویس createdAt.

لازمه؟: تیک “Required” رو بزن.

Create رو بزن.

چک کن: باید sessionId, userId, role, content, createdAt رو ببینی.

۳.۴ شماره‌ی مجموعه‌ها رو نگه دار
هر مجموعه یه شماره‌ی خاص داره که برنامه‌ت بهش نیاز داره.
برو به مجموعه‌ها: توی بخش پایگاه داده، سه تا مجموعه‌ت رو می‌بینی: کاربرها, جلسه‌ها, چت‌ها.

شماره‌ها رو بگیر:
روی کاربرها کلیک کن.

بالای صفحه، “Collection ID” (شماره مجموعه) رو می‌بینی، یه رشته‌ی بلند (مثل 67f64d80000eb41830cf). اینو کپی کن و توی یه فایل متنی بذار (مثل نوت‌پد). بنویس “شماره مجموعه کاربرها”.

برگرد به Collections، روی جلسه‌ها کلیک کن و شماره‌ش رو کپی کن. بنویس “شماره مجموعه جلسه‌ها”.

همین کار رو برای چت‌ها بکن و بنویس “شماره مجموعه چت‌ها”.

جایی امن نگهشون دار: این شماره‌ها رو گم نکن، بعداً لازمشون داری!

قدم ۴: راه‌اندازی OpenRouter
OpenRouter یه سرویس مجانیه که به رباتت کمک می‌کنه جواب‌های باهوش بده.
برو به OpenRouter: توی مرورگرت بنویس https://openrouter.ai/ و Enter رو بزن.

ثبت‌نام کن: گوشه‌ی بالای سمت راست، روی Sign Up (ثبت‌نام) کلیک کن.

یه راه انتخاب کن: با گوگل یا با ایمیل و رمز ثبت‌نام کن (مثل Appwrite).

تأیید کن: اگه با ایمیل ثبت‌نام کردی، برو توی ایمیلت و لینک تأیید رو بزن.

وارد شو: برگرد به https://openrouter.ai/، روی Log In (ورود) کلیک کن و مشخصاتت رو بنویس.

کلید بگیر:
سمت چپ، روی Keys (کلیدها) کلیک کن (شاید یه آیکون کلید باشه).

دکمه‌ی Create Key (ساخت کلید) رو بزن.

توی کادر “Name” (اسم)، بنویس کلید ربات.

روی Generate (تولید) کلیک کن.

یه رشته‌ی بلند می‌بینی که با sk-or- شروع می‌شه (مثل sk-or-abc123xyz). کپی‌ش کن و توی فایل متنیت بذار. بنویس “کلید OpenRouter”.

قدم ۵: ساخت ربات توی تلگرام
تلگرام جاییه که آدما با رباتت حرف می‌زنن. با یه حساب خاص به اسم BotFather رباتت رو می‌سازی.
تلگرام رو باز کن: برنامه‌ی تلگرام رو توی گوشیت باز کن یا توی مرورگر برو به https://web.telegram.org/.

وارد شو: اگه حسابت نداری، با شماره تلفنت وارد شو (تلگرام یه کد برات می‌فرسته).

BotFather رو پیدا کن: توی نوار جستجو بالای صفحه، بنویس BotFather و Enter رو بزن. یه حساب با تیک آبی می‌بینی. روش کلیک کن تا چت باز شه.

چت رو شروع کن: بنویس /start و Enter رو بزن (یا توی اپلیکیشن دکمه‌ی Start رو بزن).

ربات جدید بساز:
بنویس /newbot و Enter رو بزن.

BotFather می‌پرسه اسم رباتت چیه. یه اسم مثل ربات باحالم بنویس و Enter بزن.

بعد می‌پرسه نام کاربریش چیه. یه چیزی بنویس که با “Bot” تموم شه، مثل ربات_باحالم_بات. (باید خاص باشه، اگه خطا داد، یه عدد اضافه کن، مثلاً ربات_باحالم_بات123).

توکن رو بگیر: BotFather یه پیام می‌ده که توش نوشته “Done!” و یه توکن بهت می‌ده (مثل 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11). همه‌ش رو کپی کن و توی فایل متنیت بذار. بنویس “توکن ربات تلگرام”.

تستش کن: توی تلگرام، نام کاربری رباتت (مثل @ربات_باحالم_بات) رو جستجو کن و Start رو بزن. هنوز جواب نمی‌ده، ولی مطمئن شو که پیداش می‌کنی!

قدم ۶: راه‌اندازی گیت‌هاب
گیت‌هاب یه سایته که کد برنامه‌ت رو اونجا نگه می‌داری و با بقیه به اشتراک می‌ذاری. مثل یه پوشه‌ی آنلاینه. نگران نباش، همه‌چیز رو توضیح می‌دم.
برو به گیت‌هاب: توی مرورگرت بنویس https://github.com/ و Enter رو بزن.

ثبت‌نام کن: گوشه‌ی بالای سمت راست، روی Sign Up (ثبت‌نام) کلیک کن.

مشخصات رو بنویس:
یه نام کاربری انتخاب کن (مثل اسمم123).

ایمیلت و یه رمز بنویس.

یه پازل ساده حل کن که نشون بده ربات نیستی.

روی Create account (ساخت حساب) کلیک کن.

تأیید کن: ایمیلت رو چک کن، یه کد از گیت‌هاب میاد، اونو توی صفحه بنویس و مراحل بعدی رو برو (مثل انتخاب پلن مجانی).

وارد شو: برگرد به https://github.com/، روی Sign In (ورود) کلیک کن و نام کاربری و رمزی که ساختی رو بنویس.

یه مخزن بساز:
گوشه‌ی بالای سمت راست، روی دکمه‌ی + کلیک کن و New repository (مخزن جدید) رو انتخاب کن.

توی کادر “Repository name” (اسم مخزن)، بنویس ربات-چتم.

زیر “Public or Private” (عمومی یا خصوصی)، Public (عمومی) رو انتخاب کن (یعنی همه می‌تونن ببیننش).

تیک “Add a README file” (اضافه کردن فایل README) رو بزن (یه صفحه‌ی توضیحات ساده می‌سازه).

دکمه‌ی سبز Create repository (ساخت مخزن) رو بزن.

چی شد؟: حالا یه “مخزن” داری (یه جور پوشه برای پروژه‌ت) توی گیت‌هاب. اینجا کد رباتت رو می‌ذاری.

قدم ۷: ساختن تابع بدون سرور توی Appwrite Cloud
“تابع بدون سرور” همون برنامه‌ایه که رباتت رو زنده می‌کنه. توی Appwrite Cloud اجرا می‌شه و همه‌چیز رو به هم وصل می‌کنه.
۷.۱ یه تابع جدید شروع کن
برو به تابع‌ها: توی صفحه‌ی پروژه‌ت (ربات من)، سمت چپ روی Functions (تابع‌ها) کلیک کن (مثل یه موشک یا علامت کد).

تابع بساز: دکمه‌ی نارنجی Create Function (ساخت تابع) رو بزن.

مشخصات رو پر کن:
اسم: توی کادر “Name” (اسم)، بنویس تابع ربات.

محیط اجرا: کنار “Runtime” (محیط اجرا)، منوی کشویی رو باز کن و Node.js 18 (یا جدیدترین نسخه‌ی Node.js) رو انتخاب کن.

دکمه‌ی آبی Next (بعدی) رو بزن.

به گیت‌هاب وصل شو:
توی صفحه‌ی بعدی، زیر “Deployment Source” (منبع نصب)، GitHub رو انتخاب کن.

روی Connect GitHub (وصل شدن به گیت‌هاب) کلیک کن. یه پنجره باز می‌شه که باید توی گیت‌هاب وارد شی. نام کاربری و رمزی که توی گیت‌هاب ساختی رو بنویس.

گیت‌هاب می‌پرسه Appwrite می‌تونه مخزن‌هات رو ببینه یا نه. روی Authorize Appwrite (اجازه دادن) کلیک کن.

برگرد به Appwrite، زیر “Repository” (مخزن)، منوی کشویی رو باز کن و مخزن‌ت (مثل ربات-چتم) رو انتخاب کن.

دکمه‌ی آبی Create (ساختن) رو بزن.

۷.۲ کدت رو اضافه کن
رباتت به یه فایل به اسم main.js نیاز داره که خودت داری. بیایم بذاریمش توی گیت‌هاب.
برو به مخزن‌ت: توی مرورگر، برو به مخزن‌ت توی گیت‌هاب (مثل https://github.com/اسمم123/ربات-چتم). “اسمم123” رو با نام کاربری خودت عوض کن.

یه فایل بساز:
روی دکمه‌ی Add file (اضافه کردن فایل) کلیک کن (نزدیک بالای سمت راست).

از منوی بازشو، Create new file (ساخت فایل جدید) رو بزن.

اسم فایل رو بذار: توی کادر “Name your file” (اسم فایلت)، بنویس src/main.js (این یه پوشه به اسم “src” می‌سازه و “main.js” رو توش می‌ذاره).

کد رو بذار: فایل main.js رو که داری، باز کن، کدهاش رو کپی کن و توی کادر بزرگ صفحه بچسبون.

ذخیره‌ش کن:
برو پایین صفحه.

توی بخش “Commit new file” (ثبت فایل جدید)، پیام پیش‌فرض (“Create src/main.js”) رو نگه دار.

دکمه‌ی سبز Commit new file (ثبت فایل) رو بزن.

صبر کن: Appwrite این فایل رو می‌بینه و شروع می‌کنه به راه‌اندازی تابع‌ت. یه دقیقه طول می‌کشه.

۷.۳ تنظیمات رو بذار
تابع‌ت به چندتا تنظیم (بهش می‌گن “متغیرهای محیطی”) نیاز داره تا به Appwrite، OpenRouter و تلگرام وصل شه.
برو به تابع‌ت: توی Appwrite، روی Functions (تابع‌ها) کلیک کن، بعد روی تابع ربات کلیک کن.

تنظیمات رو باز کن: بالای صفحه، تب Settings (تنظیمات) رو بزن.

متغیرها رو پیدا کن: پایین‌تر برو تا بخش “Variables” (متغیرها) رو ببینی.

این متغیرها رو اضافه کن: برای هر کدوم، Add Variable (اضافه کردن متغیر) رو بزن، “Key” (کلید) و “Value” (مقدار) رو پر کن و Save (ذخیره) رو بزن:
کلید: APPWRITE_ENDPOINT | مقدار: بنویس https://cloud.appwrite.io/v1 (دقیق همینو بنویس).

کلید: APPWRITE_PROJECT_ID | مقدار: برو به صفحه‌ی اصلی پروژه‌ت (سمت چپ روی Home بزن). بالای صفحه “Project ID” (شماره پروژه) رو می‌بینی (مثل 67f6ead918e72f1f30f4). کپی‌ش کن و اینجا بذار.

کلید: APPWRITE_API_KEY | مقدار:
سمت چپ، روی Settings (تنظیمات، آیکون چرخ‌دنده) کلیک کن.

روی API Keys (کلیدهای API) بزن، بعد Add API Key (اضافه کردن کلید API).

اسمش رو بذار کلید ربات.

زیر “Scopes” (دسترسی‌ها)، تیک databases.read، databases.write و functions.read رو بزن.

روی Generate (تولید) کلیک کن، بعد کلید (مثل abc123xyz...) رو کپی کن و اینجا بذار.

کلید: OPENROUTER_API_KEY | مقدار: کلید OpenRouter رو از فایل متنیت کپی کن.

کلید: TELEGRAM_TOKEN | مقدار: توکن ربات تلگرام رو از فایل متنیت کپی کن.

کلید: USERS_COLLECTION_ID | مقدار: شماره مجموعه کاربرها رو از فایل متنیت کپی کن.

کلید: SESSIONS_COLLECTION_ID | مقدار: شماره مجموعه جلسه‌ها رو کپی کن.

کلید: CHATS_COLLECTION_ID | مقدار: شماره مجموعه چت‌ها رو کپی کن.

همه‌چیز رو ذخیره کن: پایین صفحه، دکمه‌ی Update (به‌روزرسانی) رو بزن.

قدم ۸: وصل کردن تلگرام به تابع‌ت
تلگرام باید بدونه رباتت کجاست تا پیام‌ها رو براش بفرسته. به این می‌گن “وب‌هوک”.
آدرس تابع‌ت رو بگیر: توی Appwrite، برو به Functions، روی تابع ربات کلیک کن و تب Settings رو بزن. پایین‌تر، “Function Domain” (آدرس تابع) رو می‌بینی (مثل https://67f6ead918e72f1f30f4.appwrite.global). کپی‌ش کن.

وب‌هوک رو تنظیم کن:
یه تب جدید توی مرورگرت باز کن.

توی نوار آدرس، اینو بنویس، ولی جاهایی که با < > مشخص شده رو عوض کن:

https://api.telegram.org/bot<توکن رباتت>/setWebhook?url=<آدرس تابع‌ت>

<توکن رباتت> رو با توکن تلگرامت عوض کن (مثل 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11).

<آدرس تابع‌ت> رو با آدرس تابع‌ت عوض کن (مثل https://67f6ead918e72f1f30f4.appwrite.global).

مثال: https://api.telegram.org/bot123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11/setWebhook?url=https://67f6ead918e72f1f30f4.appwrite.global

Enter رو بزن.

نتیجه رو ببین: مرورگرت باید یه چیزی مثل این نشون بده:

{"ok":true,"result":true,"description":"Webhook was set"}

اگه خطا دیدی، توکن و آدرس رو دوباره چک کن.

قدم ۹: رباتت رو تست کن
تلگرام رو باز کن: برو توی تلگرام و رباتت رو پیدا کن (مثل @ربات_باحالم_بات).

پیام بفرست: بنویس /start یا فقط بگو “سلام” و Enter رو بزن.

صبر کن: رباتت باید جواب بده! اگه جواب نداد:
توی Appwrite، برو به Functions > تابع ربات > تب Executions (اجراها).

دنبال خطاها بگرد (مثل “متغیر پیدا نشد”) و اگه چیزی اشتباه بود، تنظیماتت رو درست کن.

قدم ۱۰: پروژه‌ت رو توی گیت‌هاب به اشتراک بذار
چون می‌خوای رباتت عمومی باشه، مخزن گیت‌هابت رو قشنگ می‌کنیم و به همه نشون می‌دیم.
عمومیش کن:
برو به مخزن‌ت توی گیت‌هاب (مثل https://github.com/اسمم123/ربات-چتم).

بالای صفحه، روی Settings (تنظیمات) کلیک کن.

پایین‌تر برو تا “Danger Zone” (منطقه خطر).

روی Make public (عمومی کردن) کلیک کن و تأیید کن.
